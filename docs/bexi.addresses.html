bexi.addresses module¶

bexi.addresses.create_memo(address, incident_id)[source]¶

Create plain text memo for an address/incident pair. The memo will contain <customer_id>DELIMITER<incident_id>, this is done to have full transparency on the blockchain

Parameters:
  • address (str) – address in the format <account_id>DELIMITER<customer_id>
  • incident_id (str) – unique incident id
bexi.addresses.create_unique_address(account_id, randomizer=<function uuid4>)[source]¶

The external exchange requires a unique address, which is constructed using the bitshares account_id and some unique random string used as the customer_id.

Format: <account_id>DELIMITER<customer_id>

Parameters:
  • account_id (string, format 1.2.XXX) – bitshares account id
  • randomizer (function handle) – random string generator
bexi.addresses.get_address_from_operation(operation)[source]¶

assumes that the operation is either from or to an exchange account. the address of this operation is then returned as <exchange_account_id>DELIMITER<customer_id>

bexi.addresses.get_from_address_from_operation(operation)[source]¶

returns the from address of the given operation. if the from address is the exchange account, the address contains the customer_id, otherwise NULL string is set
