static get(*args, message=None, default=None)[source]¶

This config getter method allows sophisticated and encapsulated access to the config file, while being able to define defaults in-code where necessary.

Parameters:args – key to retrieve from config, nested in order. if the last is not a string it is assumed to be the default, but giving default keyword is then forbidden

:type tuple of strings, last can be object :param message: message to be displayed when not found, defaults to entry in ERRORS dict with the

key defined by the desired config keys in args (key1.key2.key2). For example Config.get(“foo”, “bar”) will attempt to retrieve config[“foo”][“bar”], and if not found raise an exception with ERRORS[“foo.bar”] message
Parameters:default (object) – default value if not found in config
