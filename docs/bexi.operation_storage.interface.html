bexi.operation_storage.interface.retry_auto_reconnect(func)[source]¶

This is a decorator for functions that utilize some external storage driver. Any exception that reflects a connection error triggers an automatic retry of the given function. Those exceptions are defined in BasicOperationStorage.get_retry_exceptions()

Parameters:func (function handle) – the function that will obtain retry feature
